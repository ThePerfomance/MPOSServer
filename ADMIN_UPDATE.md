### 5. **Преподаватели видят всех пользователей**

Обновлен `UserAdmin`:

- Преподаватели могут редактировать всех, кроме админов
- Преподаватели не могут удалять пользователей

## 🚀 Как использовать

### Запуск сервера

```bash
docker compose up --build
# или
python manage.py runserver
```

### Вход в админ-панель

1. Перейдите на `http://localhost:8000/admin/`
2. Войдите под пользователем с ролью `teacher` или `admin`

### Создание структуры курса (для преподавателя)

1. **Предмет**: 
   - Нажмите "📖 1. Предметы" → "Добавить предмет"
   - Название: "Математика", Описание: "Курс алгебры и геометрии"

2. **Тематический блок**:
   - Нажмите "📦 2. Тематические блоки" → "Добавить блок"
   - Выберите предмет, название: "Алгебра", Позиция: 1

3. **Урок**:
   - Нажмите "🎬 3. Уроки" → "Добавить урок"
   - Выберите блок, название: "Квадратные уравнения"
   - Привяжите видео (если есть)

4. **Тест**:
   - Нажмите "✅ 4. Тесты" → "Добавить тест"
   - Название: "Проверка по теме", Длительность: 600 (сек)

5. **Вопросы**:
   - Нажмите "❓ 5. Вопросы" → "Добавить вопрос"
   - Выберите тест, введите текст вопроса
   - Добавьте варианты ответов (отметьте правильный)
   - Добавьте рекомендации для работы над ошибками

## 📝 Для диплома

### Демонстрация системы:

1. **Покажите разницу ролей**:
   - Войдите как `admin` — полный доступ
   - Войдите как `teacher` — нет удаления, нет типов видео (только просмотр)
   - Войдите как `student` — доступ запрещён

2. **Продемонстрируйте конструктор**:
   - Откройте дашборд преподавателя
   - Пройдите по шагам создания курса
   - Покажите графики успеваемости

3. **Покажите автоматический ML-анализ**:
   - Пройдите тест с ошибками через мобильное приложение/API
   - Проверьте консоль сервера — логи ML-анализа
   - Покажите рекомендации в API ответе

### Скриншоты для диплома:

1. Дашборд преподавателя с карточками статистики
2. Конструктор курса (все 5 шагов)
3. Таблица результатов тестов с цветовыми индикаторами
4. Форма создания вопроса с рекомендациями
5. Сравнение прав доступа (admin vs teacher)
